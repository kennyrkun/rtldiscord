# Requirements
Debian based system.

## System Packages
  - git
  - cmake
  - python3
  - python3-pip
  - python3-pyaudio
  - gnuradio

## Python Packages 
  - samplerate
  - discord.py[voice]
  - numpy

# Setup
1. Install boatbod/op25 from source.
2. Create a loopback device using `sudo modeprobe snd-aloop`. Use the device # in the script.
